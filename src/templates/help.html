How do I..

Start working with PassHub for Business

In your browser, navigate to the login page of your company Passhub for Business instance.

Scan to sign-in

Login to PassHub on a desktop by scanning the QR code with WWPass PassKey app on your smartphone.

Touch to sign-in

When opening PassHub in your smartphone browser, just tap the QR-code to sign-in on this phone.

NOTE: You must have WWPass PassKey app.

If you do not have an account yet, the PassHub will send you an email with a verification link. Only users with whitelisted email addresses are allowed to create an account.

Get WWPass PassKey application

  • Get "WWPass PassKey" application on AppStore or Google Play
  • Launch the app and follow instructions to personalize your PassKey app.
    Please write down and remember your WWPass PIN: it cannot be reset
  • We recommend you to execute Passkey backup procedure. It is bound to your email address. Please follow the steps to be able to transfer the application to another device if your phone is lost or stolen, or if you upgrade to a new one.
See detailed documentation at https://qrportal.wwpass.com/documentation.html.

Access PassHub records right on my phone

Yes, it is possible. Instead of scanning the QR code just tap the QR code with your finger (Unique feature!)

Create a new password safe

A password safe is a top-level folder which contains login records, notes, files, and other folders.

Click New Password Safe and enter any safe name you like

Create a new login entry, note, file or folder

To add a new record, click the Add button.

When you're on a mobile device, tap the password safe name to get to the list of safe items.

New login entry

Fill the form shown below, where

  • Title - identifies the account entry. It is the only required field in the form.
  • Username/password - the credentials of the account. Use provided random password generator to get a strong password. Just click Generate.
  • URL - network name or IP address of the resource. URLs which start with "http://" "https://" or "www." are shown as active web links
  • Notes - specific notes, instructions, or other messages are safely encrypted and stored with each entry.

New note

Note is a special type of record to store arbitrary not structured texts. The Note dialog consists only of Title and Notes fields.

New file

You can store files in PassHub too. These may be sensitive documents you share with clients or partners, SSL certificates used widely in IT infrastructure and so on. File names and content are encrypted the same way as all other data.

First click Browse button to select a file, next use Upload button.

Copy/move entries to another safe or folder

Open a menu of the desired Entry and select Cut or Copy command

Now open a menu of a destination safe or folder, note the Paste command become active. Select Paste and the entry will be copied or moved to a new location.

Please note, Paste command is only active for 30 seconds after you clicked Copy or Cut.

Share my password safe with others

You can share your records by granting other users access to the selected password safe.

NOTE: the recipient user should already be registered in the PassHub.

In the safe menu, choose 'Share' item and provide the recipient's email address. You can edit a safe name, as it will be seen by the recipient. The safe name in your account will stay intact.

You can also select a role (read-only, editor, or administrator) for the recipient user.

  • Read-only - the user can see all the safe entries, but cannot add, modify or delete entries.
  • Editor - in addition to viewing the information, the user can add, modify or delete safe entries.
  • Safe Administrator - in addition to editor rights, the user can share the safe, change other user rights or revoke user access.

Backup and restore my records

Export your records in XML or CSV file and import them later.

Export/Backup

Select Export link under your password safe list. In the provided dialog choose one of two file formats - CSV or XML. For backup/restore purposes they are equally good. However, if you want to have exported data in human-readable form or to put them into an Excel table, use CSV.

Compatibility with KeePass

There are at least three flavors of KeePass password manager:
  • KeePass 2 for Windows
  • Keepass2Android - obviously for Android devices
  • KeepassXC - cross-platform application for MAC, Linux, and Windows

Windows and Android versions support an XML format as a primary choice. CSV implementation contains errors and is not recommended. Cross-platform KeePassXC supports CSV only, but this implementation is correct.

SUMMARY: use CSV for KeePassXC and XML for KeePass2/Windows and KeePass2andoird

Import/Restore

You can only import PAssHub or KeePass-compatible files. They have CSV and XML format.
  1. Select Import link under the password safe list
  2. Select mode: Import (puts file data into a newly created single safe) or Restore (merges file data with existing safes)
  3. Select the file you wish to upload
NOTE: Restore mode preserves all the data you have in PassHub, it can only add new records or additional versions of existing records. It never deletes or modifies existing data.

User management

Selected users may be granted PassHub Administrator role. They may access the User Management page which shows overall site resources and user activity. PassHub administrator may invite new users, delete expired accounts, grant or revoke PassHub Administrator rights.

To delete a particular user account, find the user in the table and click a red cross icon.

To toggle PassHub Administrator rights, click a green checkbox icon.

email address white list

For a user to be able to create an account in PassHub for Business, the user email should be whitelisted. This white list includes all addresses in predefined mail domains (e.g., your company domain) and a list of individual addresses. Passhub Administrator may add or delete individual addresses in the white list thus granting or revoking access to PassHub service.
